# Aurora Luxury Jewelry — E-Commerce Platform

Aurora is a premium, full-stack e-commerce application designed for an exclusive jewelry brand. Built using modern Django architecture and an elegant frontend interface, the platform offers an immersive, high-end shopping experience tailored for the modern connoisseur.

## ✨ Key Features

* **The Inner Circle (User Authentication):** Robust user registration, custom profile configurations, and production-ready password reset workflows securely handled via automated Google SMTP integration.
* **Asynchronous Luxury Showroom:** Implemented an optimized JavaScript/AJAX pipeline enabling live category filtration, seamless infinite-scroll product loading, and instant shopping vault metrics sync without rendering page reloads.

## 🛠️ Tech Stack

* **Backend:** Python / Django 
* **Frontend:** HTML5, CSS3, JavaScript (ES6 AJAX/Fetch API), Bootstrap 5.3
* **Database:** Postgresql
* **Email Service:** Google SMTP Server Integration
